This is a functional code of option price and greeks calculator. There are two files :
1.option_greek_functions : It includes all the formulae functions of black-scholes option pricing  and for greeks.
2. option_greek_calculator : It is a php file, which will do the front-end function.

Requirements : 
Python installed (preferably 2.7)
php installed (5+)
Server (preferably apache)
